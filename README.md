# DashboardKit-style Vite React Bootstrap Admin

**Tech**: Vite + React 18 + Bootstrap 5 + Bootstrap Icons + Recharts

**Features**
- Dark sidebar + white topbar (DashboardKit vibe)
- Dashboard with stat cards and a combined Bar + Line chart
- Users table (search, sort, pagination)
- Forms page with Bootstrap components

## Run
```bash
npm install
npm run dev
```
Build & preview:
```bash
npm run build
npm run preview
```
